# Repo Cleanup Plan

Do not delete anything until the user explicitly approves the exact repo path.

## Candidates

| Repo | Path | Reason | Last Commit | Remote | Recommended Action |
| --- | --- | --- | --- | --- | --- |
| <name> | <path> | <reason> | <age> | <remote/no remote> | archive/delete/keep |

## Approval Checklist

- [ ] Repo path confirmed
- [ ] No uncommitted user work
- [ ] No unique files worth preserving
- [ ] Remote/backups checked
- [ ] User explicitly approved removal

## Actions

- Archive first when unsure.
- Prefer `trash` over permanent delete.
- Never remove repos from a shared/work root silently.
